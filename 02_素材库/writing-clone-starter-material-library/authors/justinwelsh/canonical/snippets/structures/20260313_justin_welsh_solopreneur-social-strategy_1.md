## 内容

Identify audience -> Choose platform -> Set goals -> Content strategy -> Brand voice -> Visual identity -> Engage -> Analyze

## 中文转写

一人企业增长链路：精准定位 -> 锁定主场 -> 量化目标 -> 内容规划 -> 确立人设 -> 视觉心智 -> 真实互动 -> 数据复盘

## 为什么好

- 闭环结构完整
- 逻辑递进感强
- 涵盖了从品牌到执行的全流程

## 4D标签

- 哲学层：深度胜过广度，与其在所有平台表现平平，不如在一个主战场做到卓越。
- 心理层：利用结构化的“操作系统”感来对抗一人企业在面对海量任务时的“溺水感”。
- 传播层：通过80%的价值输出建立信任，20%的商业推广完成转化，构建良性增长循环。
- 社会层：社交媒体的本质是“社交”，通过真实、及时的双向互动而非单向广播来建立忠诚社区。

## 可变体方向

- 极简3步法：选对人、进对场、说对词
- SOP版：社交媒体获客标准作业流程
