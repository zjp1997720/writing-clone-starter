## 内容

1. Identify problem/solution -> 2. Define SMART goals -> 3. Break into steps -> 4. Outline tasks -> 5. Set timelines -> 6. Visualize -> 7. Review/Adjust

## 中文转写

业务路线图 7 步法：定义痛点方案 -> 设定 SMART 目标 -> 拆解里程碑 -> 罗列具体任务 -> 设定截止日期 -> 视觉化呈现 -> 动态复盘调整

## 为什么好

- 标准的线性流程，易于复制
- 涵盖了从战略（目标）到执行（任务）的全过程
- 强调了“视觉化”和“复盘”的闭环

## 4D标签

- 哲学层：一人公司的核心不是管理时间，而是通过系统化的路线图管理自律和目标。
- 心理层：利用视觉化的路线图消除大目标的畏难情绪，通过明确的检查点建立掌控感。
- 传播层：通过具体的业绩数字（$4M+）和极简的运营模式（0雇员）形成强烈的反差吸引力。
- 社会层：反映了现代劳动力向“超级个体”转型的趋势，强调个体在缺乏组织支撑时需建立自己的战略框架。
