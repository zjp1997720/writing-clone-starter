## 内容

Explain context (Unwrapping a gift) -> Breakthrough Drivers (Patterns/Originality) -> Structured Narrative Summary

## 中文转写

思想伙伴公式 = 角色设定（Fresh Eyes）+ 流程隐喻（拆礼物/由浅入深）+ 四重突破驱动力（模式识别、原创挖掘）+ 结构化输出

## 为什么好

- 将模糊的‘启发’过程标准化为可执行的算法步骤
- 强调‘由浅入深’的心理预期管理

## 4D标签

- 哲学层：AI 不应是答案的提供者，而应是思维的助产士，通过挑战而非替代来构建用户的认知肌肉。
- 心理层：利用“当局者迷”的心理，通过模拟外部视角（Fresh Eyes）揭示用户潜意识中已存在但未被察觉的智慧。
- 传播层：通过极端最高级的承诺（Greatest Ever）结合具体的生产力痛点（Writing/Clarity），激发用户的尝试欲望与转发动力。
- 社会层：在“AI 让人变笨”的社会焦虑中，提供了一种证明“AI 让人更聪明”的反直觉叙事方案。

## 可变体方向

- 深度对话结构：建立信任 -> 核心质询 -> 模式识别 -> 洞察总结
- 创作者突破模型：原始素材 -> 多维提问 -> 发现矛盾 -> 叙事重构
