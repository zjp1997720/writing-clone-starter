## 结构公式

**「现象热→本质空→我来填」四段式**

```
1. 借势：一个热门讨论/事件/活动
2. 指出空白：「大家只看现象，没人说本质」
3. 给出本质：一句话能说清的核心洞察
4. 案例验证：2-3个递进的例子（从熟悉到陌生）
5. 化解担忧：回应可能的质疑
6. 金句收束：可独立传播的结尾
```

## 为什么好

1. **借势降低阅读门槛**：读者已经对话题有认知，不用从零解释
2. **制造空白激发期待**：「没人说本质」让读者想看你怎么说
3. **本质解释体现深度**：一句话说清，显得你想得比别人透
4. **案例递进增强说服力**：从读者熟悉的场景（夫妻、孩子）过渡到你要讲的场景（AI）
5. **化解担忧消除阻力**：不让读者带着疑虑离开
6. **金句收束便于传播**：最后一句能被单独截图分享

## 4D标签

- 传播层：借势开头、信息空白、递进案例、金句收尾
- 社会层：「我比别人看得深」的身份塑造
