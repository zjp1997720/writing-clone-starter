## 内容

Identity Anatomy: Goal -> Perception -> Learning -> Action -> Conditioning -> Identity -> Defense.

## 中文转写

身份解构公式：目标 -> 感知 -> 学习 -> 行动 -> 强化 -> 身份建立 -> 身份防卫。

## 为什么好

- 将抽象的心理学过程公式化，便于理解
- 解释了为什么改变很难（因为有身份防卫机制）
- 提供了干预改变的切入点

## 4D标签

- 哲学层：人生是目标的投射，系统性改变必须从身份认知的“第一性原理”开始，而非单纯的行动修改。
- 心理层：利用“反向愿景（Anti-vision）”将潜意识的痛苦转化为显性的改变动能，克服防御性心理机制。
- 传播层：采用“1天修复人生”的极端标题策略，结合年末“新年决心”的集体焦虑节点进行爆款传播。
- 社会层：解构了从工业时代延续至今的从众心理与身份认同，倡导数字时代的个人主体性与机构化自我。
