# 风格检查清单模板

## 使用说明

这个清单在 Phase 2.1 生成,用于指导后续的骨架生成和完整草稿扩展。

## 清单内容

### 1. Form 底盘目标

- **目标**: 主判断优先由完整自然段承载，短句只用于提气、转折、落点
- **来源**: 从 style-profile.md + form-normalized-anchors/ 提取

### 2. 禁用表达清单

**绝对禁止**:

- [ ] "首先/其次/最后"
- [ ] "综上所述"
- [ ] "本文将"
- [ ] "让我们一起"
- [ ] "不难发现"
- [ ] [其他禁用表达]

### 3. 常用句式

**必须使用**:

- "先说结论:..."
- "说白了,..."
- "你想想:..."
- "这就是..."
- "区别在于:..."
- [其他常用句式]

### 4. 节奏控制规则

- **关键句处理**: [单独成段/加粗/其他]
- **段落长度**: 单段不超过[X]行
- **留白规则**: [关键观点后空一行/其他]
- **模块预算**: 全文主模块不超过[X]个；每个模块只负责一个任务

### 5. 金句密度

- **目标**: 每[X]字至少1个可独立传播的金句

### 6. 对话感要求

- **目标**: 每[X]字至少1个口语化表达
- **示例**: "你想想""说白了""先说结论"

### 7. 引号与标点统一

- **目标**: 中文正文默认统一使用 `「」`
- **禁止**: 把 `“ ”` 当成常规中文引号
- **例外**: 代码、配置项、外文原句、字段名

### 8. 信息密度要求

- **目标**: 每个主模块都必须提供新增信息
- **检查**: 不能只换一种说法重述前文观点
- **压缩规则**: 删掉后主线不受影响的段落，默认应删除或并段

### 9. 素材增强预算

- **Goldmine 使用档位**: `skip / low / medium / high`
- **默认规则**:
  - 强经历/强判断: `skip` 或 `low`
  - 方法论/教程: `medium`
  - 纯框架: `medium` 或 `high`
- **硬约束**:
  - 每个槽位最多补 1 条（`gold_sentence` 最多 2 条）
  - 总条数 <= 6
  - 素材只能补槽位，不能代替主笔

### 10. 反模式检查

**必须避免**:

- [ ] AI味(过于工整、没有情绪)
- [ ] 正确的废话(空洞无物)
- [ ] 高高在上(专家口吻)
- [ ] 论文腔(学术化表达)
- [ ] 结构性重复(同一主张被重复解释)
- [ ] 信息增量不足(文字变长，但新增价值没有同步增加)

## 评分标准

### 风格对齐度评分(140分)

1. **Form 底盘**(20分):
   - 主判断成段承载，短句只局部提气: 20分
   - 基本达标，但有1处轻微碎段: 10分
   - 正文明显碎切或短句越权: 0分

2. **禁用表达**(20分):
   - 无命中: 20分
   - 命中1-2个: 10分
   - 命中3个以上: 0分

3. **常用句式**(20分):
   - 使用3个以上: 20分
   - 使用1-2个: 10分
   - 未使用: 0分

4. **节奏控制**(20分):
   - 完全符合: 20分
   - 基本符合: 15分
   - 不符合: 0分

5. **对话感**(20分):
   - 强烈: 20分
   - 一般: 10分
   - 缺失: 0分

6. **信息密度**(20分):
   - 高：段段有新增信息，无明显重复: 20分
   - 一般：有1处轻微重复或展开过长: 10分
   - 低：同一主张被多次重讲: 0分

7. **标点统一**(20分):
   - 全程统一使用 `「」` 且无混用: 20分
   - 有1处轻微混用: 10分
   - 多处混用 `“ ”`: 0分
