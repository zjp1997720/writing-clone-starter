# 选题方法论：50选1 + 意料之外情理之中

## 核心公式

**每天上午脑爆，12点前从热点想出50个选题，只选1个。**

## 第一法则：抛弃第一直觉

**发现新观点的过程，不过是一个个抛弃旧观点的过程。**

**你第一时间、第一直觉想到的观点一定不能写！因为所有人都能想到。**

### 案例：七夕选题

**直觉会想到的选题（❌ 不能用）：**

1. 单身太久会得单身癌
2. 去年七夕你在做什么
3. 你能用专业能力为另一半做什么

**咪蒙最终选的（✅）：**

- **七夕，我在民政局排队离婚**

**为什么？**

- 七夕的关键词是：浪漫、甜蜜、美好
- 大家都往这个方向想
- 但好题目必须**自带戏剧冲突**
- 离婚 vs 七夕 = 意料之外，情理之中

---

## 第二法则：意料之外，情理之中

**内容创业必须做到八个字：意料之外，情理之中。**

| 维度         | 含义             | 检验标准               |
| ------------ | ---------------- | ---------------------- |
| **意料之外** | 读者第一眼会惊讶 | "啊？这个角度我没想到" |
| **情理之中** | 细想又觉得有道理 | "仔细一想，确实是这样" |

**如果只有意料之外，没有情理之中 = 标题党，读者会骂你**

**如果只有情理之中，没有意料之外 = 没有点击欲望**

---

## 第三法则：选题三问

在50个选题中筛选时，问自己：

### 1. 这个话题能覆盖多少人？（基数）

**越多人有共鸣的痛点越好。**

| 覆盖面 | 示例                           |
| ------ | ------------------------------ |
| **广** | 职场压力、婚恋焦虑、亲戚问工资 |
| **窄** | 某个小众行业的问题             |

### 2. 这个话题能让人生气吗？（强度）

**情绪强度决定转发率。**

| 强度   | 情绪                 | 效果   |
| ------ | -------------------- | ------ |
| **强** | 愤怒、委屈、焦虑     | 高转发 |
| **弱** | 有点不开心、有点无奈 | 低转发 |

### 3. 这个话题有"假想敌"吗？（靶子）

**有敌人的文章更容易引发站队。**

| 靶子     | 示例                     |
| -------- | ------------------------ |
| **清晰** | 渣男、奇葩老板、贱人亲戚 |
| **模糊** | 社会压力、时代焦虑       |

## 50选1操作流程

### 步骤1：热点扫描（9:00-10:00）

**每天早晨扫描：**

- 微博热搜
- 知乎高赞回答
- 朋友圈刷屏话题
- 头条热点

**重点不在于发生了什么，而在于网民在骂什么。**

### 步骤2：脑爆50个选题（10:00-11:30）

**创作团队一起脑爆，必须凑够50个。**

**技巧：**

- 同一个热点，从不同角度切入
- 例如明星出轨：
  - 角度1：渣男的惯用套路
  - 角度2：原配的隐忍
  - 角度3：小三的心理
  - 角度4：吃瓜群众的双标

### 步骤3：意料之外筛选（11:30-12:00）

**从50个中筛选，问自己：**

- 这个角度，别人能想到吗？
- 如果能想到，删掉
- 如果想不到，留下

**最后选出1个最"意料之外"的。**

## 选题来源：四级采访

选题不是凭空想出来的，而是**从采访中发现的**。

### 采访中发现的金矿案例

| 采访对象   | 发现的洞察                             | 成为的选题                     |
| ---------- | -------------------------------------- | ------------------------------ |
| 少女群     | "友情就是双标"                         | 《友情就是双标》               |
| 女生群     | "男朋友拍照技术太烂"                   | 《你男朋友为什么把你拍那么丑》 |
| 90后群     | "每换一次工作，朋友圈点赞的人都换一批" | 《你的朋友圈，是职场编年史》   |
| 商学院同学 | "正室要像小三一样活着"                 | 《正室要像小三一样活着》       |

**要点：**

- 采访时记录那些让你"咦？"的瞬间
- 那些你没想到但细想很有道理的观点
- 那些具体的、生动的细节

## 选题自检清单

在最终确定选题前，检查：

- [ ] **意料之外**：这个角度，竞品能想到吗？
- [ ] **情理之中**：细想，读者会觉得有道理吗？
- [ ] **基数够大**：这个痛点，多少人有共鸣？
- [ ] **情绪够强**：这个话题，能让人生气吗？
- [ ] **有假想敌**：这个话题，谁是"坏人"？
- [ ] **抛弃直觉**：这是我第一直觉想到的吗？（如果是，危险）

---

## 选题禁区

### 绝对不碰

1. **政治敏感** - 涉及领导人、政策、体制
2. **公共数据造假** - 可以被查证的数据
3. **时间线造假** - 可以被扒出来的细节

### 慎重考虑

1. **极端阶层对立** - 容易同时激怒底层和顶层
2. **绝对的性别对立** - 容易引发反噬
3. **涉及真实公众人物** - 容易被律师函

---

## 咪蒙的选题笨功夫

**"其实我的秘诀就是笨。笨功夫。"**

- 每天50个选题是雷打不动的
- 不是灵感驱动，是方法驱动
- 写作是有方法的，爆款是可以复制的

**任何一个好产品，都是聪明人用笨功夫做出来的。**

---

**终极原则**：选题决定了天花板。标题决定生死，但选题决定上限。一个平庸的选题，再好的标题也救不回来。
