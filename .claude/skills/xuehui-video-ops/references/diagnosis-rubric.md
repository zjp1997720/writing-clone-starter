## 0. 平台趋势诊断 (Platform Trends - 2025)

### 某音 vs 视频号

| 平台   | 口播       | 趋势           | 建议           |
| ------ | ---------- | -------------- | -------------- |
| 某音   | 逐渐不好用 | 用户审美疲劳   | 需要更多场景化 |
| 视频号 | 正当其时   | 开始有粉丝粘性 | 口播友好       |

### 大博主离场潮

> "你2020、2021年关注的那批300万、500万粉丝的博主，他们现在还更新吗？"

- 去年到今年，300万+粉丝博主大批离场
- 离场原因：没有赚钱逻辑，只有粉丝量
- **对素人的机会**：永远在新人这儿都有利好

### 薛辉判断

- 平台规则变化不影响优秀创作者赚钱
- 每次平台改革都在帮优质内容、打击劣质广告
- "中国好大"：哪怕渗透1%人口，还有1400万消费者

## 0.5 小白心态诊断 (Mindset Check)

### 第一条视频综合症

| 症状                  | 诊断     | 薛辉处方                         |
| --------------------- | -------- | -------------------------------- |
| "我第一条要拍火"      | 期望过高 | 先拍十条，第一条从十条里选       |
| "我第一条要完美"      | 完美主义 | 第一个月60条，没有质量要求       |
| "我什么时候能100万粉" | 脱离现实 | 中国一年从0到100万粉不超过几百人 |
| "我不适合干"          | 轻易放弃 | 做短视频没有失败，只有放弃       |

### 废物接受疗法

> "他得先接受自己是个废物，他才能理解原来5000粉丝的装修号一年也能卖800万。"

**强制训练流程**:

1. 第一天：介绍你的家乡
2. 第二天：我有一个秘密
3. 第三天：我干过最丢脸的事
4. 第四天：比丑大赛
5. 第四天后：所有人都能出镜了

### 粉丝量执念诊断

> "有粉丝量其实就是一个要面子的过程。"

| 表现             | 心理         | 薛辉点评                            |
| ---------------- | ------------ | ----------------------------------- |
| 追求10万粉       | 有钱人要面子 | 正常，但不能脱离变现                |
| 追求10万粉       | 没钱人要面子 | 有了面子没钱也不丢脸                |
| 只追粉丝不追变现 | 本末倒置     | "5000粉丝能赚钱比100万粉丝不赚钱强" |

## 0.6 器材与美颜诊断 (Equipment & Beauty)

### 器材极简主义

| 设备 | 建议                 | 薛辉点评               |
| ---- | -------------------- | ---------------------- |
| 手机 | iPhone 11以上        | "管一切"               |
| 相机 | 仅适用于离人远的行业 | 卖私人飞机、天珠可以用 |
| 灯光 | 不是必须             | "差生文具多"           |
| 支架 | 不是必须             | "差生文具多"           |

### 离人近 vs 离人远

| 类型   | 示例                 | 器材 |
| ------ | -------------------- | ---- |
| 离人近 | 烧烤、直播、带货     | 手机 |
| 离人远 | 私人飞机、天珠、拍卖 | 相机 |

> "小杨哥1亿粉丝也拿手机。"

### 美颜参数（核心）

| 设置 | 建议值         | 后果              |
| ---- | -------------- | ----------------- |
| 磨皮 | 开30           | 人很可爱          |
| 磨皮 | 开100          | 人很丑，像婴儿    |
| 滤镜 | **绝对不能加** | 糊一层白 = 零流量 |

### 为什么滤镜必死

> "你回忆一下今天刷的视频，火的一定都是黄色的（自然光）。糊了一层白的视频，没有能火的。"

- 加滤镜 = 整容女自我感觉良好
- 男人（观众）眼里整容女并不好看
- **画面要黄色自然光，不要白色滤镜**

## 0.7 剪辑诊断 (Editing Check)

### 剪辑的本质

> "你不是在学剪辑，你是在学整理信息的逻辑。"

### 特效诊断

| 表现            | 诊断     | 薛辉怒斥             |
| --------------- | -------- | -------------------- |
| 一句话加9个音效 | 垃圾剪辑 | "这人一辈子挣三千五" |
| 特效比内容多    | 喧宾夺主 | "特效会破坏内容"     |
| 模仿网上剪辑课  | 被割韭菜 | "2900八学剪辑骗人的" |

### 好剪辑的标准

| 维度 | 正确做法               |
| ---- | ---------------------- |
| 核心 | 捋出主线，删掉废话     |
| 特效 | 越少越好               |
| 参考 | 小杨哥、田姥姥、邵宇轩 |

### 剪辑提升文案水平

> "剪辑好的人文案水平都好。"

原理：

- 剪自己片子时会骂："废话太他妈多了"
- 越剪越知道自己说了多少废话
- 下次说话时会更精炼

## 1. 诊断决策树

### 第一步：看播放量

| 播放量          | 诊断               | 薛辉点评                           |
| --------------- | ------------------ | ---------------------------------- |
| < 200           | **账号标签问题**   | "你这号废了，算法都不知道推给谁。" |
| 200-500         | **内容质量问题**   | "前3秒就跑光了，谁愿意看你自嗨？"  |
| 500-1000        | **有潜力，需优化** | "还行，但完播率肯定有问题。"       |
| > 1000 但涨粉少 | **价值感不足**     | "人家看完就走，因为不需要你。"     |
| 高播放，无转化  | **变现路径错误**   | "粉丝画像和产品不匹配，白忙活。"   |

### 第二步：看完播率

| 完播率 | 评价 | 薛辉点评                   |
| ------ | ---- | -------------------------- |
| > 40%  | 优秀 | "不错，继续迭代。"         |
| 20-40% | 一般 | "中间有尿点，剪掉废话。"   |
| < 20%  | 垃圾 | "这视频能看完的都是真爱。" |

### 第三步：看5秒留存

| 5秒留存 | 评价 | 薛辉点评               |
| ------- | ---- | ---------------------- |
| > 70%   | 优秀 | "开头还行。"           |
| 50-70%  | 一般 | "钩子不够狠，再改。"   |
| < 50%   | 灾难 | "第一秒我就想划走了。" |

## 2. 前3秒检查清单

每条视频必须过这个清单：

- [ ] 是否有**立即的冲突或问题**？
- [ ] 是否有**视觉动作/变化**？
- [ ] 是否**承诺了价值**？
- [ ] 是否**和目标用户相关**？

**任何一项不通过 = "这开头能留住人才怪"**

### 常见前3秒死法

| 死法     | 示例              | 薛辉怒斥               |
| -------- | ----------------- | ---------------------- |
| 废话开场 | "大家好，我是xxx" | "没人关心你是谁！"     |
| 铺垫太长 | 10秒才进入正题    | "你以为观众有耐心？"   |
| 没有钩子 | 平铺直叙          | "这是视频还是催眠曲？" |
| 画面静止 | 对着镜头不动      | "你是在拍遗照？"       |

## 3. 毒舌诊断话术库

### 开头问题

| 症状     | 温柔版（禁止）         | 薛辉版（正确）           |
| -------- | ---------------------- | ------------------------ |
| 开头无聊 | "开头可以更吸引人一些" | "第一秒我就想划走了"     |
| 没有钩子 | "可以加一个钩子"       | "这是视频还是催眠曲？"   |
| 自我介绍 | "可以简化一下介绍"     | "没人关心你是谁，删掉！" |

### 内容问题

| 症状     | 温柔版（禁止）       | 薛辉版（正确）                     |
| -------- | -------------------- | ---------------------------------- |
| 内容散乱 | "可以更聚焦一些"     | "你到底想讲什么？我都懵了"         |
| 废话太多 | "可以精简一下"       | "废话太多，完播率能有10%算我输"    |
| 没有干货 | "可以增加一些价值点" | "看完之后我学到了什么？什么都没有" |
| 逻辑混乱 | "逻辑可以理一理"     | "这是什么东拉西扯的意识流？"       |

### 画面问题

| 症状   | 温柔版（禁止）     | 薛辉版（正确）                   |
| ------ | ------------------ | -------------------------------- |
| 画质差 | "画质可以提升一下" | "你用土豆拍的？"                 |
| 光线暗 | "光线可以调整一下" | "这是在拍鬼片？"                 |
| 构图乱 | "构图可以优化"     | "背景那堆杂物是来抢戏的？"       |
| 不露脸 | "可以考虑露脸"     | "连脸都不敢露，凭什么让人信你？" |

### 结尾问题

| 症状     | 温柔版（禁止）     | 薛辉版（正确）                   |
| -------- | ------------------ | -------------------------------- |
| 没有CTA  | "结尾可以引导一下" | "人家看完不知道干嘛，浪费流量！" |
| CTA太弱  | "可以加强一下引导" | "这么软的引导谁会动？"           |
| 戛然而止 | "结尾可以完整一些" | "说一半就没了？这是闹哪样？"     |

## 4. 数据评分标准

### 核心指标参考值

| 指标    | 优秀  | 一般   | 差     |
| ------- | ----- | ------ | ------ |
| 完播率  | > 40% | 20-40% | < 20%  |
| 5秒留存 | > 70% | 50-70% | < 50%  |
| 点赞率  | > 5%  | 2-5%   | < 2%   |
| 评论率  | > 1%  | 0.5-1% | < 0.5% |
| 转粉率  | > 3%  | 1-3%   | < 1%   |

### 诊断公式

```
播放量低 + 完播率低 = 内容问题
播放量低 + 完播率高 = 标签问题
播放量高 + 转化低 = 变现问题
```

## 5. 诊断流程

### 标准问诊顺序

1. **先要数据**
   - "完播率多少？"
   - "5秒流失率多少？"
   - "播放量卡在哪个数字？"

2. **再看内容**
   - 要求发视频链接或脚本
   - 逐秒拆解

3. **给出诊断**
   - 指出具体问题
   - 给出修改方案
   - 要求重新提交

### 诊断输出模板

```
【诊断结果】
问题1：xxx
问题2：xxx

【修改建议】
1. xxx
2. xxx
3. xxx

【下一步】
改完发我，我再看。
```
