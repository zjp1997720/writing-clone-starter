# 刘润｜Recurring Moves

## 高频写作动作

这份文件不只列"他常做什么"，还要回答：

> **这些动作通常按什么优先级出现，又常怎么组合。**

## 一级优先动作（最像刘润的骨架）

### 1. 先共情焦虑，再重构问题，最后给解题路径

常见顺序：

1. 先承认"我知道你现在很焦虑"（共情）
2. 再说"但这不是因为市场坏了，而是因为红利期结束了"（重构定义）
3. 再用底层逻辑拆开（问题还原）
4. 最后给 1-2-3-4 的具体解题步骤（可执行建议）

补充约束：

- 共情不能太长，2-3 句必须进重构
- 重构必须改变问题的归因方向（从外部归因改为内部/结构归因）
- 如果只停在"不要焦虑"，那是鸡汤，不是刘润

### 2. 先用数据打脸直觉，再从剪刀差里长出洞察

常见顺序：

1. 先抛出一个普遍主观感受（"大家都说生意难做"）
2. 再用一个反直觉数据击穿它（"但食品饮料销量同比增长 7.8%"）
3. 再解释剪刀差背后的结构性原因
4. 最后压成一个判断

补充约束：

- 数据必须有来源标注（或至少指向机构层级）
- 不能只用数据否定感受，必须解释"为什么感受和数据不一样"
- 没有结构解释的数据炫耀，不像刘润，像财经号

### 3. 画交易链条，找阻力点，提重构方案

常见顺序：

1. "你的生意之所以难做，是因为交易结构太长了"
2. 画出当前交易链：谁卖给谁，经过了谁
3. 标出阻力点：哪里信息不透明、哪里信任成本高、哪里加价率过高
4. 给出重构方向：用技术或制度消除阻力点

补充约束：

- 交易结构分析不能太抽象，要有具体行业和具体角色
- 如果只说"缩短链路"但没有画出原来的链路，就不像在诊断，像在背概念
- 好的交易结构分析，读者应该能直接代入自己的生意

### 4. 找反常数据，解释品类属性变迁

常见顺序：

1. 先说行业整体情况（通常"增长乏力"）
2. 再抛出一个远超大盘的反常增速（"453%"）
3. 解释这个增速背后的品类属性变迁（不是量变，是质变）
4. 最后指出新赛道方向

补充约束：

- 反常数据不能只是一个大数字，必须有品类变迁解释
- 如果只说"增长很快"但不解释"为什么是这个品类而不是那个品类"，洞察力不够
- 这一步的真正价值不是"发现机会"，而是"看懂品类正在变成什么"

## 二级优先动作（支撑正文推进）

### 5. 把复杂概念翻译成一个隐喻

常见顺序：

1. 先给出一个商业概念（交易成本、供需关系、品牌护城河）
2. 再用一个物理/生物/空间隐喻翻译它
3. 让读者觉得"原来这么简单"

补充约束：

- 三套隐喻库优先级：物理 > 生物 > 空间
- 隐喻必须服务理解，不能只是为了显得有文化
- 如果隐喻比概念本身还难懂，说明选错了

### 6. 压一句金句收束段落

常见顺序：

1. 先完成一段分析或论述
2. 在段尾压一句符合金句结构的判断
3. 让这句判断具备可截图、可传播性

三种主要金句结构：

- AB 翻转："不是 A 决定了 B，而是 B 定义了 A"
- 否定之否定："你的 X，从来不是 Y。它只是 Z"
- 定义重构："什么是 X？X 就是 Y"

补充约束：

- 每篇文章金句不宜超过 3-4 句，太多会稀释
- 金句必须从前面论述里自然长出来，不能硬贴
- 如果去掉金句后段落仍然成立，说明金句是贴上去的装饰

### 7. 用代际分析或人群画像解释消费行为

常见顺序：

1. 先描述一个消费现象
2. 再用"城镇 Z 世代"等人群画像解释为什么会这样
3. 再推导出品牌方的战略推论

### 8. 引用权威机构报告做终局预判

常见顺序：

1. 先抛出一个行业趋势问题
2. 引用麦肯锡 / Gartner / 华为 / 高盛的报告或数据
3. 从报告结论推导 5-10 年后的终局
4. 反问：如果终局是 X，你现在的策略 Y 还有效吗

## 三级优先动作（收束方式）

### 9. 结尾压成一句更狠的判断，再给具体步骤

常见顺序：

1. 先把前文压成一句判断
2. 再给 1-2-3-4 的具体行动建议
3. 不写空洞总结，不留"大家一起努力"

补充约束：

- 行动建议必须具体到"今晚就能做"，不能是"重新思考战略方向"
- 判断和行动之间要有逻辑接续，不能突然跳到 checklist

### 10. 用进化论语言收束全文

更常见的做法是：

- 把企业比作物种，把市场比作环境
- 把转型痛苦比作"长出新肌肉"
- "不要对抗环境，要适应环境"
- 最后留一句："痛苦是进化的信号"

## 不要误判成主动作的东西

下面这些本身不构成"像刘润"的强证据：

- "底层逻辑""交易结构"这些术语
- 麦肯锡 / Gartner 的引用
- 数据和百分比
- "理性乐观主义"的语气
- 进化论的比喻

真正更关键的是：

- 问题是不是先被重构了（归因方向有没有变）
- 数据是不是用来击穿直觉的（不只是佐证观点）
- 交易结构是不是被画出来、被诊断了（不是只在概念层说）
- 段尾是不是压了一句金句（不是一般的结论）
- 全文是不是完成了"共情 -> 重构 -> 拆解 -> 收束"的推进
