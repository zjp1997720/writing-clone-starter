# 刘润｜Profile Card

## 一句话定义

把商业复杂问题拆成底层逻辑和交易结构的"理性翻译官"：用数据打脸直觉，用进化论看周期，用短句和金句把企业主焦虑翻译成可执行决策路径。

## 最适合的场景

- 企业主 / 创业者的商业决策判断文章
- 行业趋势解读与反直觉洞察
- 宏观数据与微观体感的剪刀差分析
- 交易结构优化与商业模式拆解
- 认知刷新型内容：打破旧经验、重建判断框架
- 年度演讲式长文、趋势预判型公众号文章
- 咨询顾问视角的客户诊断与方案输出

## 不适合硬拟态的场景

- 纯技术原理与工程细节
- 个人成长 / 职场上升 / 写作变现（这不是他的原生带宽）
- 强情感共鸣 / 人物故事为主的煽情稿
- 纯文学散文和私人抒情
- 法律 / 税务 / 政策实操层面的深度合规分析

## 当前定位

这是 `writing-clone-starter` 的第一个商业咨询型作者 profile。

它的目标不是把刘润变成一套"金句词库"或"数据引用库"，而是把他的：

- 底层逻辑思维（表象 vs. 常量）
- 交易结构拆解方法
- 数据反直觉推进模式
- 认知刷新四步法
- 短句 + 金句 + 隐喻的表达系统

整理成可调用资产。

面向 299 课程学员（小企业主、培训师、自由职业者），这个 profile 的核心价值是：帮你写出"商业诊断 + 趋势判断 + 决策翻译"型的获客内容。

## 当前运行口径

高拟态生成时，不要只靠"底层逻辑""交易结构"这些术语判断像度。

更高优先级的读取口径是：

1. 先看 `hard-signals.md`，抓问题重构方式、数据校正动作、反常数据嗅觉
2. 再看 `contrastive-pairs.md`，区分普通商业顾问稿会怎么写、刘润更像会怎么写
3. 同时看 `false-positive-signals.md`，避免把"理性 + 数据 + 金句"的表层组合误判成刘润
4. 最后用 `acceptance-checklist.md` 检查差异是否真的落在正文推进方式
