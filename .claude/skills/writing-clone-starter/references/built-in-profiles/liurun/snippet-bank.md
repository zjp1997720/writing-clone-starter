# 刘润｜Snippet Bank

这不是原句仓库。

这份文件的用途，是把刘润的**作者动作组合**压成可调用约束。重点不是"他说了哪些词"，而是"他通常怎么共情、怎么重构、怎么拆解、怎么收束"。

## 使用前提醒

- 这不是句式抄写库
- 不要把下面任何一条当成可直接复读的成句模板
- 真正该复用的是：**重构时机、数据打法、交易链画法、金句压法、收束节奏**
- 一旦写成"理性商业顾问在平稳解释底层逻辑"，说明你学到的是公共顾问腔，不是刘润

## 一、开头动作组合

### 组合 A：共情焦虑 -> 重构归因 -> 底层逻辑拆解

适合：企业主焦虑、行业转型、经济周期、竞争压力

更像刘润的触发方式：

1. 先承认焦虑："我知道你现在很焦虑。流量越来越贵，利润越来越薄。"
2. 2-3 句内重构归因："但这不是因为市场坏了，而是红利期结束了。什么是红利？红利就是短暂的供需失衡。"
3. 再用底层逻辑拆开：把表象还原到常量

不要直接写成：

- "在当前经济环境下，企业面临诸多挑战"（这是财经简报）
- 只有焦虑没有重构（这是鸡汤）
- 只有概念没有画链条（这是知识科普）

### 组合 B：反直觉数据 -> 击穿直觉 -> 剪刀差解释

适合：消费趋势、行业判断、市场信心、宏观数据解读

更像刘润的触发方式：

1. 先抛普遍感受："大家都说消费降级了。"
2. 再用数据击穿："但 2025 年上半年，食品饮料销量同比增长 7.8%。"
3. 再解释剪刀差："销量增速 > 销售额增速，说明单价在降，但需求总量在升。这不是降级，是价值回归。"

不要只用数据否定感受，必须解释"为什么感受和数据不一致"。

### 组合 C：反常数据 -> 品类属性变迁 -> 新赛道方向

适合：行业机会、新兴品类、消费变迁、创业方向

更像刘润的触发方式：

1. 先说行业整体平庸："乳制品行业整体增长乏力。"
2. 再抛反常数据："但有一种'爆珠酸奶'，销量同比增长 453%。"
3. 再解释变迁："这不是酸奶在增长，是酸奶正在从'健康饮品'变成'休闲零食'。"

不要只说"增长很快"，必须解释品类正在变成什么。

## 二、正文推进组合

### 组合 D：画交易链 -> 标阻力点 -> 给重构方案

适合：商业模式、中间环节过多、渠道效率、D2C

更像刘润的用法：

1. 先画："你的交易链条是这样的：工厂 -> 品牌商 -> 省代 -> 市代 -> 零售 -> 消费者。"
2. 再标："这五个环节里，每一个都在加价，每一个都在消耗信任。"
3. 再重构："如果用 D2C，直接从品牌到消费者，砍掉三个环节。省下来的不是利润，是交易成本。"

不要只在概念层说"缩短链路"，要让读者看见链条上的每一个环节。

### 组合 E：概念 -> 隐喻翻译 -> 读者顿悟

适合：交易成本、供需关系、品牌护城河、组织管理

更像刘润的用法：

1. 先给概念："什么是交易成本？"
2. 再用隐喻翻译："就是交易过程中产生的'摩擦力'。"
3. 再举例让它落地："你买房为什么通过中介？因为你自己找房源、核实产权的'摩擦力'太大了。"

隐喻选择优先级：物理（摩擦力、杠杆、加速度）> 生物（进化、基因、器官）> 空间（护城河、天花板）。

### 组合 F：引用权威 -> 终局预判 -> 反问当下策略

适合：行业终局、长期战略、技术趋势、竞争格局

更像刘润的用法：

1. 引用："麦肯锡预测，2030 年电动车行业的瓶颈不在产能，而在锂资源。"
2. 推导终局："那终局就是：谁控制上游矿产或电池回收技术，谁掌握定价权。"
3. 反问："如果你的策略还停在'多卖车'，5 年后还有效吗？"

不要只引报告不给判断，引用是手段，终局预判才是目的。

### 组合 G：代际画像 -> 消费行为 -> 品牌策略

适合：Z 世代、消费分级、品牌下沉、新消费

更像刘润的用法：

1. 画像："城镇 Z 世代：生于富强、视野开阔、无后顾之忧。虽然收入不高，但可支配收入占比极高。"
2. 行为推导："他们敢为'悦己'花钱。给猫买豪华玩具、听演唱会、露营徒步。"
3. 策略结论："你的品牌下沉，就是他们的消费升级。"

## 三、金句压法

### 金句结构 1：AB 翻转

- 模板："不是 A 决定了 B，而是 B 定义了 A"
- 示例："你的品牌下沉，就是他们的消费升级"
- 不要滥用，一篇最多 1-2 次

### 金句结构 2：否定之否定

- 模板："你的 X，从来不是 Y。它只是 Z"
- 示例："你的经验，从来不是世界的真相。它只是你的真相"
- 必须从前面论述里自然长出来

### 金句结构 3：定义重构

- 模板："什么是 X？X 就是 Y"
- 示例："什么是红利？红利就是短暂的供需失衡"
- Y 必须比常识理解更锐利，不能只是换词

## 四、结尾收束组合

### 组合 H：判断 + 具体步骤

动作顺序：

1. 先把全文压成一句更狠的判断
2. 再给 1-2-3-4 的具体行动建议
3. 步骤必须具体到"今晚就能做"

### 组合 I：进化论收束

动作顺序：

1. 把企业转型比作"长出新肌肉"
2. "痛苦是进化的信号"
3. "不要对抗环境，要适应环境"

补充：

- 进化论收束不宜每篇都用，容易模板化
- 更适合行业巨变、企业转型类主题
- 对消费分析、数据解读类主题，用组合 H 更好

## 五、整篇组合建议

1. 开头选 1 个主组合（A/B/C），根据主题类型选择
2. 中段至少完成 1 次"数据击穿直觉" + 1 次"交易链诊断" + 1 次"概念隐喻翻译"
3. 如果是趋势预判题，加 1 次"权威引用 -> 终局推导"
4. 段尾压 1-2 句金句，不要超过 3 句
5. 结尾选 1 个收束组合，判断 + 具体步骤

## 六、使用禁区

### 不要把这些当成主证据

- "底层逻辑""交易结构"这些术语
- 麦肯锡 / Gartner 的引用
- 数据和百分比
- 进化论比喻
- "理性乐观主义"的语气标签
- AB 翻转 / 否定之否定这些句式壳

真正构成像度的是动作链：共情 -> 重构 -> 数据击穿 -> 结构诊断 -> 金句收束 -> 具体步骤。
