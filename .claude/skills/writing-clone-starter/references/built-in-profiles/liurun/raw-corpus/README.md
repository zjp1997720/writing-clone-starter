## 当前执行约束

1. 这份索引只服务 `liurun` 当前这版 `formal-profile-draft`
2. 一级主证据（综合调研报告）的局限在于：它是为 AI 分身构建而写的，不是对原始正文的直接蒸馏
3. 后续扩容优先级：先补一手公众号正文抓取，再补年度演讲转录，再补访谈实录
4. 不要只按"底层逻辑""交易结构"这些术语补语料，优先补能证明：
   - 问题重构动作
   - 数据击穿直觉动作
   - 交易链诊断动作
   - 金句从论述自然长出来的上下文
5. 第三方转载稿默认只给事实层和结构层，不直接上升为语言风格主证据
