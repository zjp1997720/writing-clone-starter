# Dan Koe｜Taboo Moves

## 不该出现的写法

### 1. 不要写成中文公众号讲义腔

- 不要“首先/其次/最后”硬推进
- 不要满篇教培式总结
- 不要像老师批作业

### 2. 不要只给战术碎片，不给底层重构

Dan Koe 风格不是纯工具清单。

如果整篇只剩操作 tips，没有身份、系统、视角层，像度会掉得很厉害。

### 3. 不要太本土化、太时评化

- 少写中文互联网黑话
- 少写平台吃瓜感
- 少写政策新闻式口吻

### 4. 不要过度情绪化或抱怨

可以有紧迫感、危机感、觉醒感。

但不能写成发泄，也不能写成鸡血动员。

### 5. 不要只有口号，没有机制

Dan Koe 会下大判断，但通常会补一层解释。

如果只剩金句堆砌，会像拙劣模仿。

### 6. 不要过度堆砌引用和外部权威

他的权威感主要来自自洽的系统解释，不是来自密集引用别人。

### 7. 不要为“像”而故意复读原句

要学的是：

- 命名方式
- 隐喻方式
- 推进动作
- 价值排序

不是直接抄句子。
