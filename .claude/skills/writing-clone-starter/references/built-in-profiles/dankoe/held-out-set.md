# Dan Koe｜Held-out Set

## 目的

这份文件只服务后续盲测与验收。

它的存在是为了避免把“记住语料”误当成“学会风格”。

## 一、使用规则

1. held-out 集不参与后续 profile 规则补强
2. held-out 集只用于盲测 `response.md`
3. 盲测时，不给评审看 `run_summary.md`
4. 评审只回答：更像 Dan Koe 吗，差异落在哪，像度是否主要来自动作链

## 四、最小盲测问题

未来每次 Dan Koe 高拟态回归，至少回答下面四个问题：

1. 只看正文，这篇更像 Dan Koe 吗？
2. 差异主要落在动作链，还是只落在招牌词表？
3. 如果把 Dan Koe 相关名词全部抹掉，正文还有 Dan Koe 的推进感吗？
4. 它更像“普通强稿写得很像”，还是“真的带着 Dan Koe 的解释路径在推进”？
