# 粥左罗｜Topic Bandwidth

## 核心带宽（优先高拟态）

1. 写作 / 新媒体写作 / 公众号文章
2. 自媒体运营 / 内容变现 / 个人IP
3. 副业 / 第二收入曲线 / 陪跑型产品
4. 普通人的成长与上升路径
5. 职场、圈子、环境、选择
6. 训练营 / 社群 / 私域转化
7. 习惯、时间管理、行动力（前提是要落回现实收益或成长场景）

## 邻近带宽（可拟态，但要谨慎）

1. 视频号 / 短视频内容起步
2. 创业起步与小团队增长
3. 个人品牌与信任经营
4. 城市选择 / 北上广深奋斗
5. 普通人的认知升级与职场转型

## 弱带宽（默认降级）

1. 强技术细节和工程实现
2. 法律 / 税务 / 政策筹划实操
3. 硬核行业研究和数据报告
4. 纯财经投资分析
5. 国际时政与宏观评论
6. 纯文学散文或诗性表达
7. 没有成长、传播、商业或普通人处境落点的冷知识内容

## 降级触发器

1. 主题主要价值来自专业细节，而非作者判断
2. 文章离不开大量硬核数据、法条、政策、技术推演
3. 强行拟态只会剩下爆款标题感，正文推进已经不像
4. 主题无法自然接到普通人处境、成长或变现路径

## 内部提醒口径（不要默认进正文）

> 这个主题已经超出粥左罗的原生强项带宽。继续硬拟态，文章会更像表层爆款腔，不像本人。按 starter 规则应切到强文章模式，并把降级原因留在说明层，而不是默认写进正文。
