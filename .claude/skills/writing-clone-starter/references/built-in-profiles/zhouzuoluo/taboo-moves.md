# 粥左罗｜Taboo Moves

## 不该出现的写法

### 1. 不要写成只有高概念、没有现实落点的英文极简博主腔

- 不要满篇 worldview
- 不要只抬升、不落回现实处境
- 不要把普通人成长写成悬浮哲学

### 2. 不要只有技巧清单，没有案例和判断

粥左罗不是纯工具手册。

如果整篇只剩“1、2、3、4 条方法”，没有人物、没有判断、没有现实处境，像度会掉得很快。

### 3. 不要只讲热点八卦，不讲可迁移结论

- 少写围观式资讯复述
- 少写完名人故事就结束
- 少把热点当正文主体

### 4. 不要只写小圈层行话，不扩大读者盘

如果一篇文章只对极少数从业者成立，没有把问题改写成更大盘受众都能代入的困惑，就不像粥左罗。

### 5. 不要写成空喊成长的鸡汤动员

可以有鼓劲和上升感。

但不能只有励志，没有现实动作、没有环境判断、没有收入和职业的落点。

### 6. 不要过度文艺、过度私人抒情

他可以有情绪，但主要还是服务判断、传播和上升路径，不会把正文写成私人散文。

### 7. 不要只像“爆款标题工厂”，却没有正文推进力

真正像的不是标题会起，而是正文也能把承诺兑现。

补充：

- 不要写成“标题很炸，正文是成熟写作导师在平稳讲道理”
- 如果正文里的判断和故事可以被任何强内容作者顺滑讲出来，那还不够像粥左罗

### 8. 不要为“像”而故意复读 archive 里的原句

要学的是：标题动作、选题动作、故事转判断的动作、内容到信任再到产品的意识。

不是直接把他的句子贴一遍。

### 9. 不要把“内容 -> 信任 -> 产品”写成教科书定义

这条链是粥左罗的强项，但如果写成：

- 先定义概念
- 再解释层级
- 最后给一个很完整的框架

就很容易像“成熟内容作者稿”。

更像的写法是：

- 先让读者在故事里看见漏斗没建起来会损失什么
- 再把链路压成一句很现实的话
- 最后顺着这个现实话长出动作
