# 粥左罗｜Negative Corpus

## 用途

这里只收：

- 题材很像
- 词也很像
- 但正文推进更像公共强稿 / 成熟写作老师稿

它的职责不是证明粥左罗“像什么”，而是证明他**不是什么**。

## 当前已知反例边界

1. `内容 -> 信任 -> 产品` 被完整讲成三层漏斗定义
   - 为什么是反例：更像老师在讲框架，不像从损失场景自然长出的判断

2. 中段连续两段以上平推解释市场 / 方法 / 框架
   - 为什么是反例：说明场景压判断失效，滑回公共最优写法

3. 全文过于工整、太会讲道理、过少疼感
   - 为什么是反例：会得到高质量成稿，但不够像粥左罗本人稿

## 本轮扩容目标

- 至少补入 4 篇高质量反例长文
- 每篇都要说明：
  - 为什么题材贴脸
  - 为什么推进不像
  - 哪一段最容易误判成粥左罗
