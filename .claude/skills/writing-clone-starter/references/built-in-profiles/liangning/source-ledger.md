# 梁宁｜Source Ledger

## 当前说明

这份 ledger 当前服务的是 **formal-profile-draft**，不是最终版 formal profile。

这意味着：

- 当前 formal 条目代表"第一轮已进入正式草稿资产"
- 不代表证据已经达到最终不可推翻状态
- 当前证据主要来自二级调研报告和已落地 consult agent，一手公众号正文语料仍在补强中
- 后续如果补到更强一手正文或转录稿，允许继续收紧、降级或重写 formal 判定

## 证据透明度声明

当前构建集的一个核心限制：

- 主要证据来源是 `01_项目/内容创作/咨询天团/梁宁AI数字分身调研报告.md`
- 这份报告本身是一份二级调研报告，包含了对梁宁公开内容的整理、重构和模拟
- 其中引用了 9 个外部来源，但并非原文全文转录
- `.claude/agents/consult-liangning.md` 与该调研报告基本同源，可作为内部理解一致性的旁证，但不能视为独立一手证据

因此：

- 框架和方法论层面的特征（痛点/爽点/痒点、点线面体、三层能力模型）可靠性较高
- 语言风格层面的特征（隐喻密度、句式偏好、收束方式）可靠性中等
- 具体措辞和标志语层面，需要更多一手语料验证
