# 薛辉｜Source Ledger

## 当前说明

这份 ledger 当前服务的是 **formal-profile-draft**，不是最终版 formal profile。

这意味着：

- 当前 formal 条目代表"第一轮已进入正式草稿资产"
- 不代表证据已经达到最终不可推翻状态
- 后续如果补到更强一手原文或更多播客转录，允许继续收紧、降级或重写 formal 判定

## 证据诚实声明

本 ledger 中的证据存在以下已知限制：

1. 核心证据是一份调研报告（非一手原文），存在转述风险
2. 表达碎片主要来自同一期播客（薛辉×刘思毅），跨场景复现验证不充分
3. 没有薛辉本人公众号文章原文、短视频脚本原文、直播转录原文
4. 部分特征可能更接近"公开方法论蒸馏"，而非"个人写作风格蒸馏"
