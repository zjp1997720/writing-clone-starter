# Justin Welsh｜Recurring Moves

## 高频写作动作

这份文件不只列"他常做什么"，还要回答：

> **这些动作通常按什么优先级出现，又常怎么组合。**

## 一级优先动作（最像 Justin Welsh 的骨架）

### 1. 先用第一人称数字锚点建权威，再翻译成读者可直接用的公式

常见顺序：

1. 先给一个不可忽视的数字结果（$4M+ revenue, 0 employees, 750K+ followers）
2. 立刻排除外部红利（"with zero paid ads", "since 2019"）
3. 再把路径压成可复制的公式或步骤

补充约束：

- 不能只是秀数字，2-3 句内必须翻译成"你也可以用"的路径
- 如果数字只停在背书层，没有快速接到可执行系统，像度会掉向 generic business coach

### 2. 先用痛点 / 身份共鸣开场，再用直接行动指令打破惯性

常见顺序：

1. 先描述一个创作者 / solopreneur 的具体痛点场景
2. 用"Stop X, start Y"或"Stop doing X"给出直接行动指令
3. 再在指令后面接系统或公式

补充约束：

- 这类"Stop X"不能写成泛励志口号
- 更像 Justin Welsh 的版本，X 必须是一个真实的、很多人正在犯的具体行为（比如"Stop reading about starting. Stop dreaming about starting. Just start."）

### 3. 把复杂商业问题拆成编号清单 + 每条带真实案例或具体动作

常见顺序：

1. 先把一个模糊问题定义成具体子项（7 types of solopreneurs, 8 ways to improve your brand, 7 marketing strategies）
2. 每个子项给一个标杆人物或具体操作
3. 最后把清单收束成一句可记忆的判断或公式

补充约束：

- 清单本身不是特色；很多英文博主都这么做
- 更像 Justin Welsh 的地方在于：每个子项背后都有一个真实的经营判断，而不只是信息整理
- 如果清单里只是"tip + explanation"的均匀平推，像度会滑向 generic productivity blogger

## 二级优先动作（支撑正文推进）

### 4. 用"旧范式 vs 新范式"对比建立认知势差

常见顺序：

1. 先描述当前大多数人怎么做（旧范式）
2. 指出为什么这种做法已经失效
3. 给出新的、更轻量的做法（新范式）

补充约束：

- 旧范式 vs 新范式的对比不能只停在观点层
- 更像 Justin Welsh 的版本，新范式必须附带一个具体操作（比如："Old Way: Ads, Fancy website. New Way: Content, Simple landing page"）

### 5. 把恐惧 / 阻力翻译成理性风险收益分析

常见顺序：

1. 先承认恐惧的存在（"I was nothing more than a spectator"）
2. 把恐惧拆成具体条目
3. 逐条用理性分析重写恐惧

补充约束：

- 不能只写成鸡汤式的"don't be afraid"
- 更像 Justin Welsh 的版本，会把恐惧翻译成具体的 worst-case / best-case 对比，用赔率说话

### 6. 用"内容矩阵 / 公式 / 时间分配"把抽象经营变成可操作系统

常见顺序：

1. 先指出"大多数人在 X 上没有系统"
2. 给出一个可量化的公式（30-30-30 method, 70% flexible / 30% regimented）
3. 让读者感到"这不是天赋问题，而是系统问题"

## 三级优先动作（收束方式）

### 7. 结尾不写总结段落，而是给一个明确的下一步行动

更常见的做法是：

- 给一个极简的"本周就做这一件事"
- 或者给一个 30 天挑战框架
- 不写成 checklist 总结，不写成圆满感悟

补充约束：

- 结尾最后一击最好同时带行动指令和紧迫感
- 如果结尾太完整、太像 TED Talk 收束，像度会掉

### 8. 偶尔插入"反共识判断"作为记忆锚点

这类插入通常是：

- 一句简短、锋利、容易截图转发的判断
- 它不服务结构推进，而服务记忆和传播
- 比如："Striving to be like everyone else is the fastest route to mediocrity"

## 不要误判成主动作的东西

下面这些本身不构成"像 Justin Welsh"的强证据：

- 编号清单标题
- 英文短句
- Solopreneur / personal brand 这类高频词
- 第一人称开头
- Motivational 金句

真正更关键的是：

- 第一人称数字是不是很快被翻译成可复制系统
- 清单里的每一条是不是带经营判断而不只是信息整理
- 正文是不是完成了"痛点 -> 公式 -> 行动"的推进
- 有没有 owned-audience 意识和漏斗思维
- 结尾是不是给了明确的下一步，而不是圆满总结
