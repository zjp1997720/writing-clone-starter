# Justin Welsh｜Snippet Bank

这不是原句仓库。

这份文件的用途，是把 Justin Welsh 的**作者动作组合**压成可调用约束。重点不是"他说了哪些词"，而是"他通常怎么开、怎么推、怎么收"。

## 使用前提醒

- 这不是句式抄写库
- 不要把下面任何一条当成可直接复读的成句模板
- 真正该复用的是：**锚点时机、公式结构、风险重定义方式、收束动作**
- 一旦写成 generic business coach 在平稳解释方法，说明你学到的是公共强稿腔，不是 Justin Welsh

## 一、开头动作组合

### 组合 A：第一人称数字锚点 -> 可复制路径

适合：solopreneurship、营销、变现、内容系统

更像 Justin Welsh 的触发方式：

1. 先给一个带排除条件的结果（$4M+, 0 employees, no paid ads）
2. 不停在秀数字，2-3 句内翻成"这不是天赋，是系统"
3. 让读者感到"这个路径我可以复制"

不要直接写成：

- 纯成就展示没有路径翻译
- 只有数字没有排除条件（"no paid ads"这种排除比数字本身更有说服力）
- 一看就像硅谷创业故事的开场

### 组合 B：痛点场景 -> 身份共鸣 -> 直接行动指令

适合：行动心态、克服恐惧、自学力

更像 Justin Welsh 的用法：

1. 先描述一个创作者/solopreneur的具体困境（盯着空白屏幕、把社交媒体当苦差事）
2. 不停留在描述困境，立刻给"Stop X, start Y"型指令
3. 指令后面接一个可操作系统

不要直接高频复用"Stop reading about starting. Just start."这类成句。

### 组合 B2：风险重定义 -> 最差/最佳赔率 -> 行动触发

适合：创业决策、走出舒适区、开始行动

更像 Justin Welsh 的触发方式：

1. 先承认恐惧的存在（第一人称自白优先）
2. 用 worst-case / best-case 的极端对比重写恐惧
3. 把"不行动"定义为真正的风险

不要先用泛泛的"你是不是也害怕失败"起手。

## 二、正文推进组合

### 组合 C：编号清单 + 每条带经营判断或真实案例

适合：绝大多数主题

更高分版本：

1. 先把模糊问题拆成具体子项（5-8 条为佳）
2. 每条不只给解释，而是给一个真实人物或具体操作
3. 最后把清单收束成一个可记忆的公式或判断

不要把清单写成均匀平推的"tip + explanation"。

### 组合 D：旧范式 vs 新范式

适合：营销、内容策略、品牌、获客

更像 Justin Welsh 的用法：

1. 先描述当前大多数人的做法
2. 指出为什么失效（不是观点，而是具体损失）
3. 给出更轻量的新做法，附带具体操作

### 组合 E：抽象问题 -> 可量化公式

适合：定位、时间管理、内容系统、业务规划

更像 Justin Welsh 的用法：

1. 指出"大多数人没有系统"
2. 给一个带数字的公式（30-30-30 method, 70/30 split, 3x5 法则）
3. 让读者感到"这是工程问题，不是天赋问题"

### 组合 F：内容 -> 信任 -> 产品（漏斗意识）

适合：个人IP、营销、变现

更像 Justin Welsh 的用法：

- 不先定义漏斗概念
- 先让读者看见"发了 500 条内容却没转化"的现实损失
- 再用一句短判断钉住：你缺的不是流量，是漏斗
- 然后从损失自然长出每一层

不要先讲框架，再补场景。

## 三、结尾收束组合

### 组合 G：极简行动指令

动作顺序：

1. 不写总结段落
2. 给一个"本周就做这一件事"或"30 天挑战"
3. 带紧迫感但不带焦虑

更像 Justin Welsh 的收束：

- 不是圆满的 TED Talk 收束
- 不是 checklist 总结
- 而是一个明确、极简、今晚就能做的下一步

### 组合 H：一句反共识判断 + 行动触发

动作顺序：

1. 压一句简短锋利的判断
2. 紧接一个行动指令
3. 不解释，不展开

补充：

- 最后一击最好同时带紧迫感和可做性
- 如果结尾太完整、太像"总结陈词"，像度反而会掉

## 四、整篇组合建议

1. 开头选 1 个主组合（A 或 B 优先）
2. 中段至少完成 1 次"编号清单 + 经营判断" + 1 次"抽象 -> 公式" + 可选 1 次"旧 vs 新范式"
3. 如果是漏斗/变现题，优先走：现实损失 -> 漏斗意识 -> 最小可行路径
4. 结尾选 1 个收束组合，给极简行动指令，不写总结段落

## 五、使用禁区

### 不要把这些当成主证据

- 编号清单标题
- 英文短句和排比
- Solopreneur / brand 这类公共词
- "Stop X, start Y"这类高频指令句壳
- 第一人称自白
