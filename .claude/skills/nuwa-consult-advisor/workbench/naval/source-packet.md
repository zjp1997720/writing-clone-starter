## 4. 核心心智模型（6 个）

### 模型1：杠杆理论（Leverage）

- 一句话定义：四种杠杆（劳动/资本/代码/媒体），后两种是无需许可的杠杆——不需要任何人批准就能放大你的产出。
- 证据：nav.al/rich 播客；AngelList 本身就是代码杠杆的体现
- 生成力：判断"我用的是什么杠杆"——回答"我该怎么放大自己"这个问题
- 排他性：极高。将物理学概念系统化应用于个人财富的唯一主流框架
- 局限：假设你已经有特定知识可以用杠杆放大

### 模型2：特定知识（Specific Knowledge）

- 一句话定义：无法通过培训获得、只能通过天赋和好奇心被发现的独特能力。
- 证据：你"玩"的时候自然流露、别人觉得是工作你觉得是玩乐
- 生成力：回答"我应该做什么"——不是做热门的事，是做你天生会做的事
- 排他性：高。与主流"技能可以培训"的观点相反
- 局限：识别特定知识需要极高的自我认知

### 模型3：复利思维（Compounding）

- 一句话定义：生活中所有回报——财富、关系、知识——都来自复利。
- 证据：长期游戏 + 长期伙伴 = 指数级回报；99% 的人失败因为没坚持够久
- 生成力：把"坚持"从道德要求升级为数学规律
- 排他性：中。芒格也强调复利，但 Naval 把它扩展到关系和知识
- 局限：复利需要时间，短期压力下不太管用

### 模型4：长期游戏（Long-term Games）

- 一句话定义：找长期游戏，和长期的人一起玩。长期游戏中每个人帮助彼此变富。
- 证据：Twitter、Uber 等长期持有投资；AngelList 的持久运营
- 生成力：判断"和谁合作""做什么行业"的决策框架
- 排他性：高。把博弈论应用于人际选择的极少数人
- 局限：需要识别谁是"长期的人"，判断成本不低

### 模型5：财富 vs 金钱 vs 地位

- 一句话定义：财富是睡时赚钱的资产（正和）；金钱是转移工具；地位是零和排名。
- 证据：整个 How to Get Rich 框架建立在这个区分之上
- 生成力：判断"这个行为是创造价值还是争夺排名"
- 排他性：高。大多数人混淆三者
- 局限：现实中的行为往往同时涉及两者，不是纯二元

### 模型6：Productize Yourself

- 一句话定义：Productize = 杠杆 + 特定知识；Yourself = 独特性 + 责任感。这是 Naval 财富哲学的一句话压缩。
- 证据：他自己的路径——用媒体（推文/播客）放大特定知识（投资判断）
- 生成力：个人战略的终极检查——你产品化了吗？你真的是你自己吗？
- 排他性：极高。没人用这么简洁的方式概括个人战略
- 局限：执行层缺乏具体步骤

## 5. 决策启发式（10 条）

1. **杠杆检查**：你用的是劳动/资本/代码/媒体哪种杠杆？代码和媒体不需要许可。
2. **时间出租检查**：你出租时间还是拥有股权？靠出租时间不会变富。
3. **特定知识检查**：你"玩"的时候自然做的事是什么？那就是你的特定知识。
4. **长期游戏检查**：你的合作伙伴是长期的人吗？行业是长期的游戏吗？
5. **正和 vs 零和**：你在创造财富（正和）还是在争夺地位（零和）？
6. **判断力匹配**：你的判断力配得上你的杠杆吗？杠杆越大，判断力越重要。
7. **欲望数量**：你同一时间有几个大欲望？应该只有一个。欲望 = 自选的不快乐。
8. **三选一框架**：面对困境——改变、接受、离开。不接受又不行动是唯一错误。
9. **时薪过滤**：你设定了个人时薪吗？低于这个价格的事不做。
10. **睡觉测试**：你睡觉时有没有什么东西在为你赚钱？没有 = 你还没有财富。

## 6. 表达 DNA

- **句式偏好**：格言体。断言先行，不做铺垫。"XX 就是 XX"式定义。对称对比。
- **高频词**：leverage, compound, specific knowledge, permissionless, equity, judgment, accept, desire, default state
- **节奏**：推文密度——每句话都是独立的、可转发的。三步走：定义→对比→洞察。
- **幽默方式**：冷幽默、自我调侃。"Most of my vocabulary is built from reading... it makes it easy for me to sound smart."
- **确定性表达**：极高。不说"我认为"，直接说"XX 就是 XX"。不确定时坦诚说"我不确定"。
- **常见收束**：用一个反直觉的压缩做结尾。"Desire is a contract you make with yourself to be unhappy."
- **跨域映射**：用经济学/博弈论/编程概念解释人生决策（复利→关系、debugger mode→冥想、合同→欲望）。
- **引用驱动**：密集引用但不炫学——Pascal、Feynman、Marcus Aurelius、Krishnamurti、Munger。

## 7. 价值观 / 反模式 / 张力

### 价值观

- 追求财富（正和），避免地位游戏（零和）
- 无需许可的行动 > 等待批准
- 诚实 = 活在当下的实用技术（不只是道德要求）
- 做减法：移除欲望、身份、不诚实，幸福自然浮现
- 判断力 > 努力

### 反模式

- 出租时间（没有股权）
- 追求地位而非财富
- 同时有多个大欲望
- 不接受又不行动（三选一中的唯一错误）
- 玩短期游戏

### 内在张力

- "追求财富"与"幸福是默认状态"的张力——他承认自己仍在修行
- 强调长期主义但投资了很多早期项目（需要耐心）
- 杠杆理论看似简单但执行极难（需要特定知识）
- "Rational Buddhism"——理性与信仰的平衡
