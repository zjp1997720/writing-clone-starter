# Business Handdrawn Mode

## Goal

Define what makes a body illustration feel like a **commercial hand-drawn infographic**, not just an infographic.

This reference exists because density alone is not enough.

An image can be information-dense and still feel dead.

What we want is:

- structured
- business-usable
- technical
- energetic
- clearly drawn by hand

## Core Visual Markers

The image should visibly contain:

- sketch-line quality
- slight wobble
- marker or felt-tip stroke feeling
- hand-shaped arrows and connectors
- boxes and dividers that are clear but not machine-perfect
- icons or figures with doodle energy, without becoming childish

And the hand-drawn contract should be legible through four primary signals:

1. handwritten-looking Chinese
2. visible paper / whiteboard surface feeling when appropriate
3. human irregularity in arrows, dividers, boxes, and connectors
4. overall resistance to template-slide neatness

## Typography Rules

Chinese text is one of the biggest signals.

The default should be:

- handwritten-looking Chinese
- whiteboard-marker feel
- readable but not perfectly typeset
- slight unevenness is welcome

Avoid:

- printed poster-font feeling
- slide-deck typography
- enterprise template typography
- overly neat alignment that kills human energy

## Commercial Boundary

This is not cute doodle art.

This is not classroom chalkboard comedy.

This is not childlike illustration.

It should still feel:

- rational
- commercially credible
- knowledge-first
- useful in business, strategy, framework, workflow, and concept explanation contexts

## Good Test

Ask this after generation:

`这张图像是人手画出来再整理过的吗？`

If the answer is no, the hand-drawn contract failed.

Ask this too:

`这张图会不会像模板 PPT 里直接拖出来的？`

If the answer is yes, the hand-drawn contract failed.

Ask this first-glance question too:

`我第一眼看到的是手绘知识图解，还是干净规整的信息图？`

If the first answer is “干净规整的信息图”，and only the second answer is “好像有点手绘”, the hand-drawn contract failed.

## Relationship To Default Infographic Mode

- `default-infographic-mode.md` defines density, structure, screenshot value, and article usefulness
- this file defines the human visual texture and handwritten energy

Both should be satisfied at the same time for the default body-image mode.
