# WeChat to Teleprompter

这是一个给“**公众号成稿已经有了，现在要尽快进入录制**”场景准备的轻量 skill。

## 什么时候用

- 你已经有 `草稿_final.md` 或其他成型长文
- 你直接贴了一篇完整正文
- 你现在不缺内容，只缺一个能录的视频号提词稿
- 你想减少重复解释“文章怎么转成口播稿”的成本

## 什么时候不要用

- 你只有一个模糊选题
- 你还没把文章写出来
- 你要的是爆款短视频运营建议
- 你要的是账号诊断、变现咨询、选题判断

这种情况，去用写作类 skill 或 `xuehui-video-ops`。

## 默认输出

- 默认只输出 **提词版**
- 默认附带 **5 个推荐短标题**
- 不默认输出三件套
- 原文不覆盖
- 优先落到原文同目录
- 如果原文属于“误区拆解 / 认知升级 / 非共识判断”型文章，会优先保留原文的认知冲突骨架，而不是只做字面压缩

## 它和 xuehui-video-ops 的关系

- `xuehui-video-ops`：更像短视频运营顾问，负责脚本创作、诊断、方法论
- `wechat-to-teleprompter`：更像内容转换器，负责把现成公众号稿变成能录的提词稿

## 最核心的一句话

**当瓶颈已经不是“写什么”，而是“怎么把成稿改成能录的视频号提词稿”，就该用我。**
